# txt2pub (txt2pub_1)

브라우저에서 **TXT 또는 EPUB 파일을 업로드**하면,
- **TXT만 업로드 시 → TXT 합본 1개 생성**
- **EPUB만 업로드 시 → EPUB 합본 1개 생성**
하는 간단한 합본 도구입니다.  
(모바일/PC 모두 사용 가능)

---

## ✅ 주요 기능

### 1) TXT 합본 만들기
- 여러 개의 `.txt` 파일을 업로드하면 **TXT 합본 1개**를 다운로드합니다.
- 결과 파일명: `책이름_합본.txt`

### 2) EPUB 합본 만들기
- 여러 개의 `.epub` 파일을 업로드하면 **EPUB 합본 1개**를 다운로드합니다.
- 결과 파일명: `책이름_합본.epub`

### 3) 파일명 규칙
- `책이름` = **첫 번째 업로드 파일의 확장자를 제거한 이름**
- 예: `나의책.txt, 2.txt, 3.txt` 업로드 → `나의책_합본.txt`

### 4) 중요한 제한 사항
- **TXT와 EPUB을 섞어서 업로드할 수 없습니다.**
  - (예: `a.txt + b.epub` 같이 업로드하면 처리되지 않습니다.)
- 합본은 **같은 확장자끼리만** 가능합니다.

---

## 🧭 사용 방법

1. 페이지를 열고(PC/모바일) 파일을 업로드합니다.
2. **TXT 파일만** 또는 **EPUB 파일만** 선택합니다.
3. 실행 버튼을 누르면 합본 파일이 다운로드됩니다.
4. 작업 완료 후에는 자동으로 초기화되어 다음 작업을 바로 진행할 수 있습니다.
